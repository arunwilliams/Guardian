# 🛡️ Guardian

Guardian is a modern web-based **Permit to Work (PTW) Management System** designed to simplify, digitize, and standardize the work permit process for construction, engineering, industrial, and maintenance projects.

The platform enables organizations to efficiently create, review, approve, issue, monitor, and close work permits while maintaining complete traceability and compliance with safety requirements. By replacing paper-based workflows with a centralized digital solution, Guardian improves operational efficiency, enhances communication between stakeholders, and strengthens workplace safety.

Built with a clean, responsive interface, Guardian supports multiple user roles—including Requesters, Permit Holders, Reviewers, Issuers, and Administrators—ensuring that every stage of the permit lifecycle is managed with accountability and transparency.

## Why Guardian?

- 📋 Digitized Permit to Work workflow
- 🔐 Role-based access control
- ✅ Multi-level permit review and approval
- 📎 Attachment support for JSA, SWMS, and supporting documents
- 📊 Dashboard with permit tracking and status monitoring
- 📁 Export permits and reports
- 📱 Responsive design for desktop and mobile devices
- 🔍 Complete audit trail for compliance and record keeping

Guardian is developed to support industry best practices and help organizations achieve safer, more efficient, and fully documented permit management processes.
